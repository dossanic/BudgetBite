# BudgetBite
## Frontend
### Install dependencies
Run `npm install`
### Run frontend
Navigate to /frontend\
Run `npm start` 
## Backend
### Run backend
Navigate to /backend\
Run `npm start `
### Routes for Ingredient Lists
http://localhost:3000/recipes?q=\[ingredients\]\
http://localhost:3000/missing-ingredients?recipe_id=\[recipeId\]&q=\[ingredients\]
